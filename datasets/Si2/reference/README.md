# reference results

- `si2.chk_U.amn` and `si2.chk_Udis.amn`:
    for `WannierIO.jl` tests, they are generated by

    ```julia
    using WannierIO
    chk = read_chk("../si2.chk")
    Udis = get_Udis(chk)
    write_amn("si2.chk_Udis.amn", Udis)
    U = get_U(chk)
    write_amn("si2.chk_U.amn", U)
    ```
